#高橋流簡報術#

<a src='https://www.youtube.com/watch?v=Vor6Yul7CMg'>高橋流簡報視頻</a>

### 特色 ###
* 文字簡單</br>
* 發表容易</br>
* 幽默、適合社群簡報</br>

### 功能 ###
* 組件式結構，可以輕鬆修改組件</br>
* 點擊投影片右方會撥放下一張，左方撥放前一張</br>
* 可用方向鍵或空白、回車鍵操控</br>
* 預設組件右下角有全屏按鈕，可以做出類似PPT的投影片播放</br>

---

### 誰是高橋 ###
* 製作性價比極高的簡報
* 日本Ruby協會會長
* 翻譯了Ruby的中文書
